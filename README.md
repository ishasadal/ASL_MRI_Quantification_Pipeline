# ASL_MRI_Quantification_Pipeline

This repository contains the processing pipeline developed for my MSc thesis on renal perfusion quantification using ASL-MRI.

## Pipeline overview

1. Data conversion to NIfTI
2. Motion correction
3. Kidney segmentation using U-Net
4. Cortex segmentation
5. Mask processing and registration
6. Perfusion quantification
7. Biomarker extraction and analysis

## Structure

- preprocessing
- segmentation
- cortex
- perfusion
- analysis

## Data

MRI data is not included due to privacy and size constraints.
